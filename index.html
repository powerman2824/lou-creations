About Lou

Lou’s Creations showcases handmade Lichtenberg wood-burning art and resin work. This site is a portfolio—Lou isn’t selling online yet, but you can send a message if you’re interested in a similar custom piece.

  • Style: Fractal “lightning” burns
  • Location: Rhode Island
  • Shop buddy: Lucky the Boxer 🐾

How the “lightning” happens

Lichtenberg patterns are created by electricity traveling through the grain, forming branching fractals. Resin can be used to emphasize the paths and add color.
